# app-aves

**app-aves** es una aplicación web interactiva desarrollada en React que muestra información sobre aves de Chile. Los usuarios pueden explorar un catálogo de aves, ver sus detalles, y navegar entre diferentes páginas para obtener información adicional. Además, los usuarios pueden suscribirse al boletín de novedades mediante un formulario.

## Características

- **Interfaz limpia y fácil de usar**: Navega entre un catálogo de aves y visualiza información detallada.
- **Página de detalles de aves**: Al hacer clic en una tarjeta de ave, se muestra más información, incluyendo su imagen ampliada.
- **Formulario de suscripción**: Los usuarios pueden suscribirse a un boletín de noticias proporcionando su nombre, apellido y correo electrónico.
- **Paginación de aves**: Las aves se muestran en páginas, con un máximo de 10 aves por página.

## Tecnologías utilizadas

- **React**: Librería para construir la interfaz de usuario.
- **Vite**: Herramienta de construcción rápida para React.
- **MUI (Material UI)**: Framework para componentes de interfaz de usuario.
- **React Router**: Manejo de rutas dentro de la aplicación.
- **API pública de Aves de Chile**: API para obtener datos de aves en Chile.
- **GitHub y Git**: Para el control de versiones y gestión del código fuente.
